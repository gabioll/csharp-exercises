# Exercício 03 - Sistema de Funcionários em C#

## Descrição
Programa em C# com menu interativo para cadastro e consulta de funcionários.

## Funcionalidades
- Cadastrar nome e telefone
- Pesquisar funcionário por nome
- Listar todos os funcionários cadastrados
- Menu com opções até o usuário sair

## Conceitos
- Arrays
- Estrutura de repetição (while e for)
- Condicionais (if/else)
- Métodos
- Manipulação de strings
