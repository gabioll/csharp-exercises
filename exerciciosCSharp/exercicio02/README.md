# Exercício 02 - Matriz de Notas em C#

## Descrição
Programa que lê as notas de 5 alunos (3 provas cada) usando matriz.

## Funcionalidades
- Armazena as notas em uma matriz 5x3
- Exibe uma tabela com as notas
- Calcula o total de pontos por aluno
- Calcula a média de cada aluno

## Conceitos
- Matrizes (array bidimensional)
- Estrutura de repetição (for)
- Entrada e saída de dados
